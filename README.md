# pico-imu
IMU based on Raspbery Pi Pico W and MPU-9250

## Connections
| Pico          | MPU9250       |
| ------------- | ------------- |
| GP 4          | MISO          |
| GP 5          | CS_PIN        |
| GP 6          | SCK           |
| GP 7          | MOSI          |
